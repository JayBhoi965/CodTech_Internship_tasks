# File Handling Utility in Java

## Overview
This project is a simple **File Handling Utility** developed in Java as part of the **CODTECH Internship Task**.  
The program demonstrates basic file operations such as:

- Writing data into a text file
- Reading data from a text file
- Modifying existing file content

The application uses Java I/O classes and exception handling to perform all operations efficiently.

---

# Features

- Create and write into a text file
- Read data from the file
- Modify existing file content
- Console-based output
- Exception handling for errors

---

# Technologies Used

- Java
- FileWriter
- FileReader
- BufferedReader
- Scanner
- Exception Handling
